# worldclock
Terminal ui worldclock. **WIP**. blocked by chrono::zoned_time support on clang.

## Building

```bash
git clone https://github.com/notshriram/worldclock.git
cd worldclock
cmake -Bbuild
cmake --build build
./build/worldclock
```
